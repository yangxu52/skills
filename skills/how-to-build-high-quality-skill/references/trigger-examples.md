# Trigger Examples

Use these examples to sanity-check whether ` how-to-build-high-quality-skill ` should be invoked.

## Should Trigger

- "Create a reusable skill from this internal guide."
- "Rewrite this SKILL.md so it triggers more reliably."
- "Turn this article into a proper skill package with references and assets."
- "Review this draft skill and tell me whether the description will under-trigger."
- "Should this logic live in SKILL.md or in references/scripts?"
- "Add gotchas and validation guidance to this skill."
- "Improve this skill so it behaves more like a production-ready package."

## Should Not Trigger

- "Write a prompt for summarizing meeting notes."
- "Explain how skills work in general."
- "Help me complete this one-off task instruction."
- "Write a README for my project."
- "Refactor this Python file."
- "Review this SQL query for performance."
